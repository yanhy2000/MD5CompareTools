# 工具 - MD5文件校验器

> 工具系列慢慢拓展中..

这是一个简单的小工具，使用**hashlib**库获取文件md5值，使用拓展库**tkinterdnd2**编写UI，可拖拽文件到软件上也可以点击按钮选择，来对比文件md5值

# 使用

需要安装**tkinterdnd2**库：
```
pip install tkinterdnd2
```
然后在cmd内可直接执行程序：
```
python main.py
```

也可以打包使用，需要安装cx_Freeze：
```
pip install cx_Freeze
python setup.py build
```
